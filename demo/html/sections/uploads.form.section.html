Description:

These attributes are processed during hydration and are used to pull upload data from $_FILES

The data type MUST be array. The only way it can not be an array is if another hydration attribute after it converts the data into something else.

The optional parameter transpose can be used to turn multi-uploads file array into a more user friendly format by transposing them.

Examples:

No files are saved anywhere. They are uploaded to your own computer's temp directory and discarded after the script runs.
#[FileUpload]
#[FileUpload]
#[FileUpload(transpose: true)]
