Description:

Here are some examples showing how a Struct's tailor method is used to alter an object's public properties using various attributes.

Tailoring attributes are processed in order from top to bottom.

Side Note: One of the more useful combinations is using Trim followed up by UseDefaultOnEmpty.

Usage:

$Obj->tailor();

Examples:

Processing String values
#[Trim]
#[UseDefaultOnEmpty]
Default: 'test'
#[StrReplace('World', 'You')]
Field has spaces
#[Trim]
#[UseDefaultOnEmpty]
Default: 'I was empty'
#[StrReplace(
[ 'H', 'W' ],
[ 'Hello', 'World' ]
)]
#[Trim]
