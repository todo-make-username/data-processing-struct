Description:

Hydration happens when an array of data is converted and assigned into an object's public properties using the Struct's hydrate method. The brings up the question, what if the data needs to be pre-processed before being assigned to the property? That is where the hydration attributes come into play.

Hydration attributes run pre-processing on the value before it is assigned to the property. Either by transforming the value, or running pre-validation checks.

Important Note: Data is automatically converted between certain types. This can be turned off if needed. Please reference the docs for more info.

Usage:

Hydrate an object's public properties with $_POST data:

$Obj->hydrate($_POST);
